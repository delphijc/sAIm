# Doc-Gen Plugin (`!doc`)

Generates Word (.docx) and PowerPoint (.pptx) documents from markdown content sent through Discord. Uses a multi-message session model where content is buffered across messages, then generated on demand.

## Commands

| Command | Description |
|---------|-------------|
| `!doc create report "Title"` | Start a Word document session |
| `!doc create slides "Title"` | Start a PowerPoint presentation session |
| `!doc done` | Finalize and generate the document |
| `!doc cancel` | Cancel the active session |
| `!doc convert <path>` | Convert a markdown file on disk to DOCX |
| `!doc list` | List previously generated documents |
| `!doc help` | Show help text |

## Workflow

1. Start a session: `!doc create report "My Report"`
2. Send markdown content across one or more messages
3. Finalize: `!doc done` — generates the file and uploads it to Discord

### Markdown Support

During a session, the following markdown is supported:
- `# H1`, `## H2`, `### H3` headings
- `- bullet` lists
- `1. numbered` lists
- `**bold**`, `*italic*`, `` `code` ``
- `---` horizontal rules

## Document Types

### DOCX Reports
- Title page with centered title and generation timestamp
- Full markdown-to-docx conversion with proper heading hierarchy
- Numbered list support with proper indentation
- Inline formatting (bold, italic, code spans)

### PPTX Presentations
- Dark PAI-themed aesthetic (dark background, purple accents)
- Title slide with date and "Generated by Sam PAI" footer
- Each `# H1` / `## H2` heading becomes a new slide
- Bullet points from list items and paragraphs
- Slide numbers and overflow handling (max 8 bullets per slide)

## Storage

Generated documents are saved to:
```
.agent/skills/discord-remote-control/service/data/doc-gen/
```

Files are named: `<safe-title>_<timestamp>.docx|pptx`

## Dependencies

- `docx` (npm) — Word document generation
- `pptxgenjs` (npm) — PowerPoint generation
